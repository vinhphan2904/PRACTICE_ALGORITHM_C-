# Competitive Programming

A collection of my solutions and notes for competitive programming problems from various online judges. This repository documents my learning journey, problem-solving process, and algorithm practice.

## 🎯 Goals

* Improve problem-solving skills.
* Prepare for competitive programming contests.
* Build a well-organized archive of algorithms and data structures.
* Record ideas, mistakes, and lessons learned after solving each problem.

## 📂 Repository Structure

```text
CompetitiveProgramming/
│
├── Codeforces/
├── CSES/
├── VNOI/
├── LeetCode/
├── AtCoder/
├── Templates/
└── Notes/
```

Each problem is organized as follows:

```text
Problem/
├── README.md      # Problem summary, idea, complexity, notes
└── main.cpp       # Solution
```

## 📖 What Each Problem Contains

* Time & Space Complexity
